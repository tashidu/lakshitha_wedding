# Lakshitha & Sandani Wedding Website

A beautiful, simple, and responsive wedding invitation website for Lakshitha & Sandani.

## Features
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **RSVP Form**: Integrated with Google Sheets for easy guest management.
- **Google Maps**: Built-in map to the venue (Sooriya Resort and Spa).
- **Background Music**: Autoplaying background music with play/pause controls.
- **Countdown Timer**: Counting down to the big day!

## Setup
Just host these static files (`index.html`, `style.css`, `script.js`, `song.mp3`) on any platform like Netlify, Vercel, or GitHub Pages.
